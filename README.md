Утилита на языке Python для коррекции исходного массива координат, выведенных из дизайнерской программы, к реальным координатам на заготовке, после замера и воода 4 крайних точек, с учетом поворота, сдвига и масштабирования.
Исходные координаты загружаются из текстовых файлов, преобразованные данные также сохраняются в текст.
